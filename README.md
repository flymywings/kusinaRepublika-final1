# kusinaRepublika
Kusina Republika: A Filipino Recipe Website

Kusina Republika is a simple, elegant recipe website that showcases classic Filipino dishes. Each recipe page includes a hero banner, ingredients, instructions, nutrition info, embedded cooking videos, and chef’s tips — all wrapped in a clean, warm theme inspired by Filipino home cooking.

Features:
1. Modern Hero Section
Each recipe includes:
- Full-width hero image
- Dark overlay for readability
- Large dish title
- Prep/Cook/Serving info badges
2. Clean Recipe Layout
- Ingredients list
- Step-by-step instructions
- Summary description
- Chef's Tips box
- Nutrition facts
- Video tutorial
3. Design System
- Warm beige background (#fff5e6)
- Poppins font
- Rounded cards with soft shadows
- Mobile-friendly responsive layout
4. Featured Recipes Carousel
- A JavaScript-powered carousel on the Recipes page.
